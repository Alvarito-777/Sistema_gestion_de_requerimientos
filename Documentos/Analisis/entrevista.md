| Pregunta | Hallazgos principales |
|----------|------------------------|
| **P1. Área o departamento** | Operaciones/Producción lidera con **34.5%**, seguido por Gerencia/Administrativo (**22.4%**) y Tecnología/Sistemas (**20.7%**). |
| **P2. Medio más utilizado** | WhatsApp domina con **67.2%**. La comunicación “de palabra” sigue con **48.3%**. El correo queda relegado. |
| **P3. Pérdida de info en WhatsApp** | El **55.2%** está totalmente de acuerdo en que se pierde información crítica por usar WhatsApp como canal principal. |
| **P4. Importancia de registro electrónico** | El **55.2%** considera “Demasiado importante” contar con un registro inmutable para auditoría. |
| **P5. Incertidumbre del estado actual** | Percepción dividida: **27.6%** cree que sí hay claridad, pero **27.6%** y **17.2%** muestran incertidumbre. |
| **P6. Canal de actualización preferido** | El **69%** prefiere notificaciones directas al celular o correo; **17.2%** prefiere tablero visual tipo semáforo. |
| **P7. Gestión de datos faltantes** | El **79.3%** quiere que el sistema impida enviar solicitudes incompletas. |
| **P8. Organización de solicitudes** | El **46.6%** prefiere priorización por urgencia; **36.2%** por orden de llegada. |
| **P9. Alcance de solicitudes formales** | El **41.4%** opina que cualquier empleado debería poder radicar solicitudes directamente a Gerencia. |
| **P10. Expectativa de tiempo de respuesta** | El **53.4%** espera respuesta formal entre **1 y 3 días hábiles**. |
| **P11. Información mínima de rechazo** | El **50%** exige “todas las anteriores”: motivo, fecha/hora, responsable y notificación automática. |
| **P12. Filtro de solicitudes para Gerencia** | Deben pasar por Gerencia: Nuevos recursos (**65.5%**) y cambios en procesos/políticas (**62.1%**). |
| **P13. Delegación de respuestas** | El **65.5%** acepta que un coordinador o jefe responda en nombre de Gerencia. |
| **P14. Visibilidad y privacidad del área** | El **63.8%** prefiere privacidad estricta: solo solicitante y Gerencia deben ver la solicitud. |
| **P15. Requisitos para Control Interno** | El **39.7%** pide visibilidad total; **29.3%** prioriza conocer quién decidió y el motivo. |
| **P16. Corrección de errores en envíos** | El **50%** quiere poder editar la solicitud mientras no haya sido aprobada. |
| **P17. Evidencia de trabajo permitido** | El **56.9%** prefiere un PDF con rastro electrónico como respaldo legal. |
| **P18. Evidencia de trabajo finalizado** | El **55.2%** exige PDF con rastro electrónico para cerrar formalmente un requerimiento. |
| **P19. Casos reales de pérdida por WhatsApp** | Reportan retrasos, solicitudes ignoradas, descargos laborales y reprocesos por falta de canal oficial. |
| **P20. Riesgos hipotéticos identificados** | Riesgos: saturación de chats, desconexión digital e interrupciones en cadena de suministros. |
| **P21. Diseño ideal del sistema** | Preferencia por plataforma tipo CRM/ERP con tickets únicos, estados claros e inmutabilidad. |
